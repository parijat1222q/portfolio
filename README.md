# Parijat Biswas - Portfolio

Welcome to my personal portfolio website! It is built with React and features a modern, responsive design with smooth animations.

## Features

-   **Responsive Design**: Fully responsive layout that looks great on desktop, tablet, and mobile.
-   **Dark/Light Mode**: Toggle between dark and light themes (persisted in local storage).
-   **Smooth Animations**: Entrance and scroll animations powered by **Framer Motion**.
-   **Interactive UI**: Tabbed Resume section and filterable Portfolio gallery.
-   **Contact Form**: Functional contact form integrated with **EmailJS**.
-   **Notifications**: Toast notifications for user interactions using **React Toastify**.

## Tech Stack

-   **Frontend**: React.js
-   **Styling**: Vanilla CSS, Responsive Grid
-   **Animations**: Framer Motion
-   **Icons**: React Icons (Remix Icons)
-   **Form Handling**: EmailJS
-   **State Management**: React Hooks (`useState`, `useEffect`, `useRef`)
-   **Utils**: `use-local-storage`, `react-tabs`
